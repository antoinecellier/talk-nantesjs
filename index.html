Javascript everywhere

Avec react on va avoir une nouvelle façon de concevoir. Lorsque l’on travail avec ReactJs la première chose à faire c’est oublier : oublier le DOM, le HTML… on reste dans le code javascript, même pour des composants qui ont une finalité graphique.

Exemple


var HelloWorld = React.createClass({

	render: function() {
  	      return React.createElement('div',
		{ className: 'hello-world' },
		'Hello world!' );
	}

});

React.render(React.createElement(HelloWorld),
	     document.getElementById('app'));


<div id="app">
	<div class="hello-world">Hello world!</div>
</div>
Donc là ok on utilisé seulement du javascript mais on risque de perdre en visibilité au niveau du code.

Virtual DOM

Implémentation légère du DOM

Mettre à jour le DOM de la manière la plus efficace

    • Le concept derrière ReactJs est de travailler sur un DOM virtuel
    • Sur le vrai DOM dont les accès en lecture et surtout en écriture sont très lent comparés à des traitements en mémoire
    • Mise à jour optimisée du DOM réel
